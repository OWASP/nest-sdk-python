# ProjectLevel

Enum for OWASP project levels.

## Example Usage

```python
from owasp_nest.models import ProjectLevel

value = ProjectLevel.OTHER
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `OTHER`      | other        |
| `INCUBATOR`  | incubator    |
| `LAB`        | lab          |
| `PRODUCTION` | production   |
| `FLAGSHIP`   | flagship     |
